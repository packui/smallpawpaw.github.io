开发工具 - MySQL安装篇

2018-03-10
开发工具
本文总阅读量次

一、下载MySQL

MySQL官网：https://www.mysql.com/

二、安装MySQL

本文以mysql-5.5.27-winx64.msi为例进行安装，其实不管哪个版本，安装本质都是相通的

1、双击运行mysql-5.5.27-winx64.msi，安装前面部分都是傻瓜式安装方式比较简单

2、安装MySQL之后，会继续进入MySQL配置设置，如下：

选择进行详细配置还是标准配置，这里选择“详细配置”

选择安装的MySQL作为服务器还是开发使用，这里选择开发使用

选择多功能类型数据库、仅支持事务类型数据库还是非事务数据库，这里选择“多功能类型”

存储引擎的表空间配置，默认下一步即可

设置MySQL并发连接数，我们选择第一个，表示假定20个连接的模式，可以根据自己的需要设置

设置MySQL端口，默认3306即可

设置MySQL的字符集，这里选择第二个 将utf8作为MySQL默认的字符集

设置MySQL服务的名字、默认MySQL即可，将“Include Bin Directory in Windows PATH”勾选上，将MySQL的bin添加到系统环境变量中去，方便我们后期在命令行窗口操作MySQL

设置root账户的密码（自己要记牢），然后Next

点击“execute”表示执行

当出现下图，四个都打上对勾，表示安装配置成功，点击“finish”完成全部安装配置

3、验证MySQL是否安装成功

在命令行窗口下，输入: mysql -uroot -p，然后回车，输入刚刚配置的root账户的密码，出现如下界面，表示安装确实成功了

4、接下来就可以快乐的玩玩MySQL了！

完
